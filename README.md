# SETUP

use python 3.11.9 and verify

```bash
python3 --version
```

## Make a virtual environment and activate it

```bash
python3 -m venv venv
source venv/bin/activate
```
## For Windows

```bash
Set-ExecutionPolicy Unrestricted -Scope Process
python3 -m venv venv
venv\Scripts\activate
```

## Install packages

```bash
pip install -r requirements.txt
```

# Run the script

```bash
python3 main.py
```
## Stop the script

press ''q''
